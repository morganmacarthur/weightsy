# Roadmap

## Phase 0 - Compliance Baseline

- Define consent and retention requirements.
- Define disclaimers and non-diagnostic positioning.
- Define minimal PII policy.

## Phase 1 - MVP

- Phone sign-in and timezone.
- Daily reminders.
- Inbound parsing and entry storage.
- Basic trend chart and history list.

## Phase 2 - Retention

- Streaks and lightweight goaling.
- Smart reminder timing.
- Weekly summary messages.

## Phase 3 - Expansion

- Additional metrics.
- Coach/program integrations.
- API export integrations.
